# 部署指南

## 本地快速启动

### 1. 环境要求
- Python 3.12+
- pip 包管理器

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 启动服务

启动后端：
```bash
uvicorn main:app --reload
```

启动前端（新终端窗口）：
```bash
streamlit run streamlit_app.py
```

访问：`http://localhost:8501`

## 学校部署

### 最小需求
- Python 3.12 环境
- 4GB 可用内存
- 无需特殊网络配置

### 多用户部署

如需多个学生同时使用，建议：

1. 配置 Streamlit 多用户支持
2. 为后端配置 Gunicorn（生产级WSGI服务器）

```bash
pip install gunicorn
gunicorn -w 4 -b 0.0.0.0:8000 main:app
```

### 数据安全

- 所有计算在本地完成，无云端上传
- 可在内网部署，无需外网连接
